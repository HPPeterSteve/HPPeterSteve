### Sobre mim

Olá, eu sou o **Steve Homer** (HPPeterSteve).

Sou desenvolvedor focado em programação de baixo nível, com especial interesse em sistemas operacionais, segurança e performance. Gosto de entender o que realmente acontece "debaixo do capô" e construir ferramentas que sejam ao mesmo tempo **poderosas e enxutas**.

Não sou fã de soluções mágicas. Prefiro código honesto, legível e que resolva problemas reais sem adicionar complexidade desnecessária.

### Como eu penso

- Código bom não é só o que funciona. É o que humanos conseguem ler, manter e confiar.
- Segurança não é adicionar camadas infinitas. É reduzir a superfície de ataque com elegância.
- Menos é mais — especialmente em sistemas críticos.

### Meu Projeto Principal

**Komodo-Secure** — Um sandbox hardened para Linux.

É o meu principal projeto no momento. O foco é criar um ambiente de execução **extremamente isolado**, usando as melhores features do kernel Linux:

- User, Mount e PID Namespaces
- Pivot Root (mais seguro que chroot)
- Seccomp-BPF com política restrita
- Capability dropping completo
- Limites de recursos anti-DoS

Tudo isso com o mínimo overhead possível.
