### Sobre mim

Olá, eu sou o **Steve Homer** (HPPeterSteve).

Sou desenvolvedor focado em programação de baixo nível, segurança de sistemas e performance. Gosto de construir ferramentas que realmente entregam controle profundo, isolamento forte e código enxuto.

Não sou fã de soluções "mágicas". Prefiro entender o que está acontecendo debaixo do capô e construir coisas honestas.

### Como eu penso

- Bom código é aquele que humanos conseguem ler, entender e manter.
- Segurança não se faz adicionando camadas infinitas. Se faz reduzindo a superfície de ataque com elegância.
- Menos é mais — especialmente quando o assunto é segurança crítica.

### Curiosidades sobre mim

- Sou curioso, autodidata, gosto de prestar atenção aos mínimos detalhes.
- Eu era uma pessoa bem extrovertida, mas escolhi ser introvertido pois isso me faz bem, gosto de desenhar e no futuro quero fazer jogos...

### Meu Projeto Principal

**Komodo-Secure** — Um sandbox hardened para Linux.

É o meu principal projeto no momento. O objetivo é criar um ambiente de execução **extremamente isolado** usando as melhores features do kernel Linux:

- User, Mount e PID Namespaces
- Pivot Root (mais seguro que chroot)
- Seccomp-BPF com política rigorosa
- Capability dropping completo
- Limites de recursos anti-DoS

Tudo projetado para ter o **mínimo overhead possível**, mantendo alta segurança.
